# Interactive-buffoonery Tap

## How do I install these formulae?

`brew install interactive-buffoonery/tap/<formula>`

Or `brew tap interactive-buffoonery/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "interactive-buffoonery/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
